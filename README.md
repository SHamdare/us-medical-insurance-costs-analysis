# U.S. Medical Insurance Cost Analysis

This project explores a dataset of medical insurance costs in the United States to understand the factors that influence insurance charges.

## Dataset

The dataset contains information about individuals including:

- Age
- Sex
- BMI (Body Mass Index)
- Number of children
- Smoking status
- Region
- Insurance charges

## Objectives

The goal of this analysis is to explore how different factors affect insurance costs.

Key questions explored:

- Do smokers pay higher insurance charges?
- How does BMI affect insurance cost?
- Does age influence insurance charges?
- Do different regions have different average insurance costs?

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Files

| File | Description |
|-----|-------------|
| insurance.csv | Dataset used for analysis |
| us-medical-insurance-costs.ipynb | Jupyter notebook containing the analysis |
| README.md | Project documentation |

## Key Insights

Some findings from the analysis include:

- Smokers tend to have significantly higher insurance costs.
- Insurance charges increase with age.
- Higher BMI often correlates with higher medical insurance charges.
